# Светильник многофункциональный Н4
- Светильник с настройкой 256 цветов и 16 градаций яркости (сохранение 8 цветов в памяти)
- Два (три) световых эффекта: "Градиент" и "Радуга"
- Цветомузыка 
- Часы
- Будильник-рассвет
